## GE-7とは何か（7バンドグラフィックイコライザーの基本）

BOSS GE-7は、音の周波数帯域を7つに分けて個別に音量を調整できる**グラフィックイコライザー（グライコ）**です。「EQ」と略されることも多く、各帯域のスライダーを上下に動かすことで音の太さ・明るさ・抜け感を細かくコントロールできます。

7つの帯域は以下のとおりです：

| スライダー | 周波数 | 音への影響 |
|-----------|--------|-----------|
| 1 | 100Hz | 低音の重さ・ドン感 |
| 2 | 200Hz | 低中域の膨らみ |
| 3 | 400Hz | 音の厚みや籠り感 |
| 4 | 800Hz | 中域の存在感 |
| 5 | 1.6kHz | 音の押し出し感・前に出る感 |
| 6 | 3.2kHz | 高中域の抜け感 |
| 7 | 6.4kHz | 高域の煌めき・シャリ感 |
| LEVEL | 全体 | 全帯域の音量を一括調整 |

## イコライザーがブースターになる仕組み（レベルスライダーの役割）

GE-7にはLEVELスライダーが独立して備わっています。このスライダーを上げると**全帯域の音量を一括で増幅**することができます。つまり、すべての帯域スライダーをフラット（中点）のまま、LEVELだけ上げれば**クリーンブースターとまったく同じ動作**になります。

さらに、GE-7では帯域調整とブーストを同時にできるため、「音量を上げつつ、音の特性も変える」高度なブーストが可能です。これがGE-7がブースターとして特に重宝される理由です。

## ブースターとしての使い方

### ①レベルスライダーだけ上げるクリーンブースト設定

```
全帯域スライダー：中点（フラット）
LEVEL：+10〜+15dB
```

最もシンプルな使い方です。音色を変えずに純粋に音量を上げます。「ソロだけ音量を上げたい」「バンドの音量が上がった時に追随したい」という場面に最適です。

### ②中域（800Hz・1.6kHz）を持ち上げてTS系的なブーストをかける設定

```
800Hz：+5〜+8dB
1.6kHz：+3〜+5dB
400Hz：±0〜-3dB（籠り感を減らす場合はやや下げる）
LEVEL：+5〜+10dB
```

TS9（Tube Screamer）のミッドハンプを意図的に再現するセッティングです。中域が押し上がることで、バンドアンサンブルの中でギターが前に出てきます。TS9を持っていなくてもGE-7でTS系のブーストが作れます。

### ③不要な帯域をカットしながらブーストするEQブースト設定

```
100Hz：-5〜-8dB（不要な低音を削る）
200Hz：-3dB（籠り感を減らす）
800Hz：+5dB（中域を持ち上げる）
1.6kHz：+3dB（抜けを出す）
LEVEL：+8〜+12dB
```

低音域のモコモコした成分をカットしながら中域を持ち上げることで、**バンドの中で最も聴こえやすい帯域に音を集中させる**アプローチです。大人数のバンドや、ドラム・ベースの音が大きい環境で特に効果的です。

## GE-7ならではのメリット

**音作りの自由度が最大：** BD-2やTS9は音の傾向が固定されていますが、GE-7は自分で自由に帯域を設計できます。自分のギター・アンプの特性に合わせた「カスタムブースト」を作れるのはGE-7だけです。

**カットもできる：** ブースターとして使う際に「余計な帯域を削る」ことで、音の純度を上げながらブーストできます。これはBD-2やTS9にはできない芸当です。

**ノイズ対策のヒント：** GE-7はノイズが出やすいと言われることがあります。対策としては、①電源アダプターを使用する（電池よりノイズが少ない）、②高品質なパワーサプライ（エフェクター用の電源装置）を使用する、③ペダルボードの中で他のデジタルエフェクターから離して配置するなどが効果的です。
