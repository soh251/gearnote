## エフェクターボードとは？

エフェクターボード（ペダルボード）とは、複数のエフェクターペダルを1枚のボードにまとめて持ち運びやすくしたものです。ケーブルをつなぎ直す手間が省け、ライブ本番でも素早く踏み替えられます。

しかし「どのエフェクターを何台買えばいいの？」「つなぐ順番はどうするの？」という疑問を持つ初心者は多いです。このガイドでは、**予算別の最適構成**と**接続順の基本**を初心者向けに解説します。

## エフェクターの接続順の基本

エフェクターを複数つなぐとき、**接続する順番がサウンドに大きく影響します**。基本的な順番は以下の通りです：

```
ギター
→ ①チューナー
→ ②コンプレッサー（持っている場合）
→ ③歪み系（オーバードライブ / ディストーション / ファズ）
→ ④モジュレーション系（コーラス / フランジャー）
→ ⑤空間系（ディレイ → リバーブ）
→ アンプ
```

**なぜこの順番なのか？**
- チューナーを先頭に置くと、ミュート機能で静かにチューニングできる
- 歪みの前にコンプを置くと音量が均一になり、歪みが綺麗にかかる
- 空間系は歪みの後に置くと「歪んだ音が広がる」自然なサウンドになる

## 初心者がよくやる失敗3つ

### 失敗①：一気にたくさん買いすぎる

「とりあえず全部欲しい！」と最初から多くのエフェクターを買ってしまうのはNG。それぞれのエフェクターを使いこなすには時間がかかります。**まずミニマムプランから始め、必要になったら追加する**のが正解です。

### 失敗②：接続順を間違える

歪みをリバーブの後に接続するなど、順番を間違えると音が汚くなります。基本の接続順（チューナー→歪み→空間系）を守りましょう。

### 失敗③：エフェクターだけに頼りすぎる

エフェクターは演奏の補助であって、演奏の代わりにはなりません。まずはギターそのものの演奏を練習し、エフェクターはその上に乗せるものと考えましょう。
